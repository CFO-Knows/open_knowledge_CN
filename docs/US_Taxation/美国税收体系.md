# 美国税收体系

美国依据税收对象，将企业分为两种类型：

* C-Corporation: C型公司，独立的纳税实体
* Pass-Through Entities: 转嫁实体，企业本身不纳税
  1. 个体户 Sole Proprietorship
  2. 合伙企业 Partnership
  3. S-Corporation

C-Corp就好比一个实心的水桶（Bucket），所有的公司收入都会流到这个水桶里，IRS会直接对这个水桶征税，即企业所得税。缴税后，剩下的钱还在水桶里，如果老板想把钱拿出来，老板需要再交一次个人所得税，这就是双重征税。

Pass-Through好比一个空心管道（Pipe），企业赚到的钱不会停留在管道里，而是直接流过这个管道，流到企业老板的个人口袋里。IRS假装这个管道不存在，直接对老板的个人口袋征税，只征收一次个人所得税。如果公司股东有好几个，那么钱会按照约定的比例，分别流进几个股东的个人口袋里。

Pass-Through包括个体户、合伙企业、S型公司，一般LLC都被归类为Pass-Through。一个Pass-Through可能要交的税包括：（以加州为例）

小企业通过QuickBooks记账，然后通过Quickbook或Gusto发工资，到年底需要请CPA来报税。CPA的工作主要包括：

* 清理烂摊子，要3-6小时，检查一整年的账目中是否有记错的内容
* double entry表有很多借贷项目，记好后quickbook上就能生成三大报表（这里如果记错的话就会生成错误）
* invoice，形式不固定
* 每个季度要预缴一个估计值，这一步不需要严谨的推算，年底多退少补；最好就是多缴，至少缴到9成以上，国税局最希望的就是每个季度都多缴了一点点，然后年底有退点

会计报表=》税务报表（每年一次）

* 税表的格式是固定的，50个会计科目mapping到税表的15个科目
* 税务调整，要2-4小时，因为会计账的利润不等于报税的利润，CPA必须进行关键的税务调整（财税差异，小型企业忽略不计，很少会出现这种情况，目前来看是比较好调整的）
* 最后CPA用价值数钱美元的专业税务软件，比如Lacerte（高端税务软件，买liscence，在公司内部去用）, ProConnect（这个online就能用，很好用，都是quickbooks的），把经过以上两步调整后的最终数字，输入到所有相关表格中，比如Form 1065, Form 568, Schedule K-1
* 最后，由一位高级经理或合伙人，花1-2小时来审阅所有文件，确保没有错误，也没有触发IRS审计的red flags。CPA会花时间向您解释您为什么需要缴税，为什么会退税，并让您在电子授权书（Form 8879）上签字。（就是CPA解释然后公司签字，中小企业一般看一下，主要关注缴税的额度，一般没问题就签字了）
